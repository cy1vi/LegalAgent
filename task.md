第一个任务是构建面向刑事案例的结构化知识提取管道。该任务以 CAIL2018 数据集为基础，针对裁判文书中非结构化的案情描述，设计一套模块化的信息抽取 Schema，包括适用于所有案件的核心字段（如罪名、刑期、是否自首、是否赔偿等）以及按具体罪名动态适配的扩展字段（如涉案金额、伤害等级、作案手段等）。通过大语言模型（LLM）结合提示工程或微调策略，自动从文本中精准提取这些量刑相关因素，并最终建成一个高质量、可检索、可分析的结构化法律案例知识库，为后续的数据建模与智能问答提供可靠的数据支撑。

第二个任务是在结构化数据基础上实现案例聚类与量刑因素的层级建模。通过对提取后的结构化案例进行特征工程处理，采用 HDBSCAN 等密度聚类算法对同类罪名下的案件进行无监督聚类，识别出若干具有代表性的“案例原型”（archetypes），例如“轻伤+赔偿+自首→缓刑”这类典型判决模式。同时，结合统计分析方法（如回归分析、SHAP 值等），针对不同罪名分别构建量刑因素的重要性排序模型，明确哪些情节对判决结果影响最大，从而揭示司法实践中量刑的关键维度，为后续智能体的推理和解释提供依据。

第三个任务是开发基于 ReAct 框架的法律对话智能体。该智能体能够模拟法律专家的思维过程，在与用户多轮交互中主动推理并调用工具：当用户输入信息不完整时，系统能动态生成引导式问题（如“是否有自首情节？”）进行信息补全；随后调用加权混合检索模块，综合关键词匹配、向量相似度和结构化条件过滤，从案例库中找出最相关的判例；最后，结合聚类结果与量刑规则，合成一份可解释的量刑参考意见（如“类似案件平均刑期为18个月，考虑到自首，可能减轻至12个月左右”），实现专业、透明、有依据的智能法律咨询。

第四个任务是建立完整的系统测评体系并完成优化完善工作。在集成前述模块的基础上，搭建端到端的系统原型，并设计覆盖多种罪名、不同信息完整度的测试用例，从多个维度对系统性能进行量化评估：包括知识抽取的准确率、检索结果的相关性（如 MRR、NDCG）、对话任务的成功率，以及刑期预测的误差（如 MAE、RMSE）。根据评估结果，对 Schema 设计、聚类粒度、ReAct 提示模板等关键环节进行迭代优化，最终形成一个功能完整、逻辑严谨、具备实际应用价值的法律智能辅助系统，并撰写配套的技术文档与使用说明。


（1）CAIL2018 数据集（Chinese AI and Law Challenge 2018）
CAIL2018 是由清华大学自然语言处理实验室发布的大规模中文法律数据集，数据来源于中国裁判文书网，涵盖罪名预测、法条推荐、刑期预测等核心司法智能任务。该数据集具有规模大、标注规范、法律要素完整等特点，是当前法律智能研究领域的标准基准数据集之一。本课题将基于 CAIL2018 数据集构建和训练法律判决预测与智能推理模型，为模型的实验验证和性能评估提供可靠的数据支撑。
（2）中国裁判文书网
中国裁判文书网是最高人民法院统一发布裁判文书的权威平台，包含大量真实、公开的司法判决文书，覆盖刑事、民事、行政等多种案件类型。该平台为法律人工智能研究提供了真实司法语境下的文本数据来源。本课题在数据理解与任务设计阶段，将以裁判文书的结构特征和表述方式作为重要参考，以增强模型在真实司法场景下的适用性与泛化能力。
（3）ReAct: Synergizing Reasoning and Acting in Language Models
该论文提出了 ReAct 框架，将“推理（Reasoning）”与“行动（Acting）”有机结合，使大语言模型能够在推理过程中动态调用外部工具或知识源。该方法为构建具备可解释推理能力的智能系统提供了新的范式。本课题将借鉴 ReAct 的思想，在法律任务中探索“推理—检索—决策”相结合的智能判案流程，以提升系统的逻辑一致性与可解释性。
（4）Retrieval-Augmented Generation for Knowledge-Intensive NLP Tasks
该论文系统性提出了检索增强生成（RAG）框架，通过将外部知识检索模块与生成模型相结合，有效缓解大模型在知识密集型任务中的事实幻觉问题。法律任务高度依赖准确的法条和案例知识，因此本课题将参考 RAG 思路，引入案例检索或法条检索机制，提高法律推理结果的准确性与可信度。
（5）Legal Judgment Prediction via Topological Learning
该研究从拓扑学习角度建模法律判决问题，探索案件之间的结构关系与相似性，对传统基于文本特征的判决预测方法形成有益补充。该工作为本课题在案件相似度建模、结构化特征融合等方面提供理论参考，有助于提升模型对复杂法律关系的建模能力。
（6）LangChain 开源框架
LangChain 是面向大语言模型应用开发的开源框架，支持链式推理、工具调用和多模块协同。该框架能够有效组织复杂的 LLM 应用流程。本课题将参考 LangChain 的设计思想，用于构建法律推理流程与多模块协同的系统架构。
（7）LlamaIndex（原 GPT Index）
LlamaIndex 是面向大语言模型的数据索引与管理框架，支持非结构化文本的高效组织与检索。该工具适用于法律文书和法条等长文本数据的管理。本课题将在知识组织和检索模块中参考其数据建模与索引方式。
（8）Elasticsearch 搜索与分析引擎
Elasticsearch 是成熟的全文检索和分析引擎，广泛应用于文本检索与相似度搜索场景。本课题可利用其高效的检索能力构建案例或法条检索模块，为法律智能推理提供底层支撑。
